# Note_Taker

## Description

A simple note taking app that user can add, save and delete notes

## Installation Instructions 

 * Clone Repo to local machine 
 
 * Navigate to server.js file in Terminal or Command prompt 
 
 * Run command "node server.js"
 
 * In your browser's nav bar type "localhost:3000" then press enter 
 
                  or 
                  
 * Click on the Heroku Deployed Link below to start app

## Github Repo Link

 https://github.com/Mstarr85

## Heroku Deployed Link

 https://mstarr85-note-taker.herokuapp.com/

 ## Instructions

 * Click on Get Started
 
 * Write a title for your note in "Note Title" field
 
 * Write note in the "Note Text" field
 
 * Click disk symbol button that will appear to save note
 
 * After note is created and saved there will be a red trash can symbol to the right of each note.  Click on trash can to        delete note.
 
 * You will notice that if you refresh the page all the information stays there.  That is because it was saved to the local      storage.
